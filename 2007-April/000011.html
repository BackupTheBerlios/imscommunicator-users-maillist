Hi Kiran,
First of all, thanks for trying IMS-Communicator.
 
About the questions in your first email:
 
1. You don't need to compile the project if you don't want to change the code. There are binary releases (for windows and linux platforms) available at https://developer.berlios.de/project/showfiles.php?group_id=7991 which are ready to run (this way you only require the JRE 1.5). Just download them, edit the properties in the ims-communicator.xml file according to your network and user profile and run the shell script (ims-communicator.sh if in linux or ims-communicator.bat if in windows). For more details, please refer to the User Manual (https://developer.berlios.de/docman/?group_id=7991) or submit any questions that you might have.
 
2. If you want to compile, you can easily import the IMS-Communicator project to your Eclipse workspace. 
 
3. And yes, to COMPILE you will need JDK 1.5. 
 
But once again, if you just want to try the IMS-Communicator, download the binary releases instead.
