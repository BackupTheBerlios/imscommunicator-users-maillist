Hi Vin,
your problem is due to the fact that you are using JDK 6, and ims-communicator is suppose to run under JDK 5.
But if you inspect the build.xml, you can see that in the line 274 you have "<javac (...)  target="1.5" </javac>"
Try to change it for "target=1.6".
Also, under Eclipse, don't forget to change the project environment to Java 6.0, if not already done (Project > Properties > Java Build Path).
Let me know if you manage to compile it.
